# 🍕 Pizza Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project focuses on analyzing pizza sales data using Power BI to uncover key business insights and sales trends. The goal is to transform raw sales data into an interactive dashboard that helps stakeholders understand revenue performance, customer ordering behaviour, and product popularity.

The dashboard provides KPI metrics and visual reports to support data-driven decision-making.

---

## 🎯 Problem Statement

A pizza company wants to analyze its sales performance and gain insights into customer purchasing patterns.

The objective is to answer the following business questions:

### KPI Requirements

1. **Total Revenue**

   * Calculate the total revenue generated from all pizza sales.

2. **Average Order Value**

   * Determine the average amount spent per order.

3. **Total Pizzas Sold**

   * Calculate the total quantity of pizzas sold.

4. **Total Orders**

   * Count the total number of orders placed.

5. **Average Pizzas Per Order**

   * Calculate the average number of pizzas sold in each order.

---

## 📊 Dashboard Requirements

### 1. Daily Trend for Total Orders

Analyze how orders vary across different days of the week.

### 2. Monthly Trend for Total Orders

Identify monthly ordering patterns and peak sales periods.

### 3. Percentage of Sales by Pizza Category

Understand which pizza categories contribute the most to overall sales.

### 4. Percentage of Sales by Pizza Size

Analyze customer preferences for pizza sizes.

### 5. Total Pizzas Sold by Pizza Category

Compare sales volume across different pizza categories.
